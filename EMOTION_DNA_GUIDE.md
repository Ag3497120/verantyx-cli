# 感情DNAシステム
Emotion DNA System

---

## 🎯 概要

**Gemini的洞察に基づく感情DNA実装**

感情はDNAに「完成品」として記載されているのではなく、以下の3層構造で記載されています:

1. **ホメオスタシスの閾値** (あるべき状態)
2. **神経伝達物質による評価の極性** (良い/悪いの判定)
3. **全同調への強制割り込みルール** (感情の湧き上がり)

---

## 🧬 DNAの3層構造

### **第一層: ホメオスタシスの閾値**

DNAに記載されている「あるべき状態」の目標値と許容範囲。

```jcross
生成する DNA.ホメオスタシス閾値 = {
  "体温": {
    "目標": 37.0,
    "許容下限": 36.0,
    "許容上限": 38.0,
    "臨界下限": 35.0,  # これを下回ると死
    "臨界上限": 40.0   # これを上回ると死
  },

  "エネルギー": {
    "目標": 100.0,
    "許容下限": 30.0,   # これを下回ると「空腹」
    "許容上限": 100.0,
    "臨界下限": 0.0,    # これで死
    "臨界上限": 150.0
  },

  "痛み": {
    "目標": 0.0,
    "許容下限": 0.0,
    "許容上限": 10.0,   # 軽い痛みまで許容
    "臨界下限": 0.0,
    "臨界上限": 100.0   # 激痛で死ぬ可能性
  }
}
```

**重要**: これらの閾値は遺伝的に決まっている。学習によって変化しない。

特定の条件で全ノードが強制的に特定のモードに同調する。

```jcross
  },
  "優先度": 10,  # 最優先
  "リソース配分": {
    "逃走準備": 1.0,
    "学習": 0.0,       # 学習停止
    "探索": 0.0,       # 探索停止

  "色": "赤（警告）"
}
```

## 🎨 記憶に「色」を付ける

経験を記憶する際、その時の感情を「色」として記録。

```jcross
定義する 記憶に色を付ける 受け取る [経験ID] = {
  # 経験を取得
  取得する 記憶.経験 から 経験ID として 経験

  # その時の感情を色として付与
  経験.色 = 現在の感情.色
  経験.感情 = 現在の感情.支配的感情
  経験.感情強度 = 現在の感情.感情強度

  # 神経伝達物質の状態も記録
  経験.神経伝達物質 = {
    "ドーパミン": 神経伝達物質.ドーパミン,
    "ノルアドレナリン": 神経伝達物質.ノルアドレナリン,
    "セロトニン": 神経伝達物質.セロトニン
  }

  返す 経験
}
```

**効果**:
- 「赤い記憶」= 恐怖を伴った経験 → 近づかない
- 「黄色い記憶」= 喜びを伴った経験 → 近づく
- 「青い記憶」= 悲しみを伴った経験 → エネルギー回復後に再試行

## 💡 重要な洞察

### **1. 感情は「回路図」として遺伝する**

DNAに記載されているのは:
- ❌ 「恐怖という感情そのもの」
- ✅ 「どういう条件でドーパミンを出すか」という回路図
- ✅ 「どういう条件で全ノードを逃走モードに同調させるか」というルール

### **2. 3層の階層構造**

```
第一層: ホメオスタシス閾値
  ↓ (偏差を検出)
第二層: 神経伝達物質の放出
  ↓ (評価の極性を決定)
第三層: 感情の割り込み
  ↓ (リソース配分を強制変更)
全ノードの同調
```

### **3. 感情 = リソース配分パターン**

- **恐怖** = 逃走リソースへの全同調
- **喜び** = 探索・学習リソースへの全同調
- **怒り** = 攻撃リソースへの全同調

感情は「気持ち」ではなく、**システム全体の動作モードの切り替え**。

### **4. 優先度による強制割り込み**

複数の感情候補がある場合、**優先度が最も高い感情**が発火。

```
優先度:
  恐怖: 10 (最優先)
  怒り: 9
  好奇心: 7
  悲しみ: 6
  喜び: 5
  安心: 3
```

**例**:
- 恐怖 + 喜び の両方が発火条件を満たす場合 → 恐怖が勝つ（優先度10 > 5）
