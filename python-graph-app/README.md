# Python グラフアプリケーション

Pythonで作成したインタラクティブなグラフ可視化アプリケーションです。matplotlib と tkinter を使用して、データを簡単にグラフ化できます。

## 主な機能

### 📊 6種類のグラフタイプ
- **折れ線グラフ**: 時系列データや連続データの可視化
- **棒グラフ**: カテゴリ別の比較
- **散布図**: 2変数の相関関係の可視化
- **円グラフ**: 割合の表示
- **ヒストグラム**: データの分布を表示
- **箱ひげ図**: データの統計的な分布を可視化

### 🎯 インタラクティブな操作
- データポイントの追加・削除
- リアルタイムでグラフ更新
- グラフタイトル、軸ラベルのカスタマイズ
- グリッド表示/非表示の切り替え
- 凡例の表示/非表示

### 💾 データ管理
- **保存**: データをJSON形式で保存
- **読込**: 保存したデータを復元
- **画像保存**: グラフをPNG/PDF/SVG形式で保存

### 🖱️ 便利なツール
- ズーム機能
- パン機能
- グラフの保存・リセット
- データリストの表示

## 必要な環境

- Python 3.7以上
- tkinter（通常Pythonに標準搭載）
- matplotlib
- numpy

## インストール

### 1. 必要なパッケージをインストール

```bash
pip install -r requirements.txt
```

または個別にインストール：

```bash
pip install matplotlib numpy
```

### 2. アプリケーションを起動

```bash
python graph_app.py
```

または実行権限を付与して直接実行：

```bash
chmod +x graph_app.py
./graph_app.py
```

## 使い方

### 基本的な使い方

1. **アプリケーションを起動**
   ```bash
   python graph_app.py
   ```

2. **データの追加**
   - X値、Y値、ラベルを入力
   - 「追加」ボタンをクリック
   - 自動的にグラフが更新されます

3. **グラフタイプの変更**
   - ドロップダウンメニューから好きなグラフタイプを選択
   - 即座にグラフが更新されます

4. **グラフのカスタマイズ**
   - タイトル、X軸ラベル、Y軸ラベルを変更
   - グリッド表示のオン/オフ
   - 凡例表示のオン/オフ

### データの管理

#### データの保存
1. 「保存」ボタンをクリック
2. ファイル名を入力（.json形式）
3. データとグラフ設定が保存されます

#### データの読み込み
1. 「読込」ボタンをクリック
2. 保存したJSONファイルを選択
3. データとグラフ設定が復元されます

#### 画像の保存
1. 「画像保存」ボタンをクリック
2. ファイル形式を選択（PNG, PDF, SVG）
3. 高解像度（300 DPI）で保存されます

### サンプルデータ

アプリ起動時に自動的にサンプルデータが読み込まれます：
- 1月〜6月の月別データ
- 折れ線グラフで表示

サンプルデータをクリアして、自分のデータを入力できます。

## トラブルシューティング

### matplotlibが見つからない

```bash
pip install matplotlib
```

### tkinterが見つからない（Linux）

```bash
# Ubuntu/Debian
sudo apt-get install python3-tk

# Fedora
sudo dnf install python3-tkinter

# macOS（通常は不要、Homebrewの場合）
brew install python-tk
```

### 日本語が表示されない

macOSの場合、Hiragino Sansフォントが使用されます。
他のOSの場合は、`graph_app.py`の以下の部分を修正してください：

```python
# Windowsの場合
plt.rcParams['font.sans-serif'] = ['MS Gothic', 'Yu Gothic']

# Linuxの場合
plt.rcParams['font.sans-serif'] = ['Noto Sans CJK JP', 'DejaVu Sans']
```

### グラフが表示されない

- データが入力されているか確認
- グラフタイプが適切か確認
- コンソールにエラーメッセージが出ていないか確認

## カスタマイズ

### グラフの色を変更

`update_graph()`メソッド内の色指定を変更：

```python
# 棒グラフの色
self.ax.bar(labels, y_data, color='coral', alpha=0.7)  # 'steelblue' から 'coral' に変更

# 散布図の色
self.ax.scatter(x_data, y_data, c='red')  # 'steelblue' から 'red' に変更
```

### デフォルトのウィンドウサイズを変更

```python
self.root.geometry("1400x900")  # 幅x高さ
```

### グラフのDPIを変更

```python
self.figure = Figure(figsize=(8, 6), dpi=150)  # 100 から 150 に変更
```
