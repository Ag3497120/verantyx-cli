# クイックスタート - 1分で始めるグラフアプリ

## ステップ1: 必要なパッケージをインストール（30秒）

```bash
cd python-graph-app
pip install -r requirements.txt
```

または：

```bash
pip install matplotlib numpy
```

## ステップ2: アプリを起動（10秒）

```bash
python graph_app.py
```

## ステップ3: 使ってみる（20秒）

### すぐに試せること

アプリが起動すると、**すでにサンプルデータが表示されています**！

1. **グラフタイプを変更**
   - 左上のドロップダウンから選択
   - 「棒グラフ」「円グラフ」など選んでみてください

2. **データを追加**
   - X値: `7`
   - Y値: `30`
   - ラベル: `7月`
   - 「追加」ボタンをクリック

3. **画像を保存**
   - 「画像保存」ボタンをクリック
   - ファイル名を入力して保存

## よく使う機能

### データ入力
```
X値: 数値を入力
Y値: 数値を入力
ラベル: 好きな名前を入力
→ 「追加」ボタン
```

### グラフの種類
- 折れ線グラフ（デフォルト）
- 棒グラフ
- 散布図
- 円グラフ
- ヒストグラム
- 箱ひげ図

### カスタマイズ
- **タイトル**: グラフのタイトルを変更
- **X軸/Y軸**: 軸のラベルを変更
- **グリッド**: チェックボックスでオン/オフ

### データ管理
- **保存**: JSON形式でデータを保存
- **読込**: 保存したデータを読み込み
- **画像保存**: PNG/PDF/SVG形式で保存

## トラブル時

### エラーが出る場合

```bash
# matplotlib が見つからない
pip install matplotlib

# numpy が見つからない
pip install numpy

# tkinter が見つからない（Linux）
sudo apt-get install python3-tk
```

### グラフが表示されない

- データを追加してください
- 「クリア」→「追加」で再描画

## 5分で作れるグラフ例

### 例1: 月別売上グラフ

```
1月: X=1, Y=100
2月: X=2, Y=150
3月: X=3, Y=130
...
```

グラフタイプ: 棒グラフ

### 例2: 試験の点数分布

```
データをY値だけ入力:
85, 92, 78, 88, 95, 72, 80, 90
```

グラフタイプ: ヒストグラム

### 例3: 市場シェア

```
製品A: Y=35
製品B: Y=28
製品C: Y=20
製品D: Y=17
```

グラフタイプ: 円グラフ
