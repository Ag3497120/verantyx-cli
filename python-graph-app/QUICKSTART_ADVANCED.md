# 🚀 高度なグラフアプリケーション - クイックスタート

## 5分で始めるガイド

### ステップ1: インストール（1分）

```bash
# 必要なパッケージをインストール
pip install dash dash-bootstrap-components plotly pandas scipy numpy
```

### ステップ2: アプリを起動（30秒）

```bash
# アプリケーションを起動
python advanced_graph_app.py
```

次のようなメッセージが表示されます:

```
============================================================
🎨 高度なグラフアプリケーション
============================================================
🌐 アプリケーションを起動しています...
📍 URL: http://127.0.0.1:8050/
⚡ ブラウザで上記URLを開いてください
🛑 終了するには Ctrl+C を押してください
============================================================
```

### ステップ3: ブラウザで開く（10秒）

ブラウザで以下のURLを開きます:

```
http://127.0.0.1:8050/
```

### ステップ4: 試してみる（3分）

#### サンプルデータを使ってみる

1. **サンプルデータを読み込む**
   - 「📁 サンプルデータ読込」ボタンをクリック
   - 自動的に100日分のデータが生成されます

2. **グラフを切り替える**
   - 「グラフの種類」ドロップダウンをクリック
   - 「📈 折れ線グラフ」「📊 棒グラフ」「🌐 3D散布図」などを試してみる

3. **統計分析を実行**
   - 「📊 基本統計」ボタンで統計量を表示
   - 「🔗 相関分析」ボタンで相関係数を表示
   - 「📈 回帰分析」ボタンで線形回帰を実行

4. **グラフを操作**
   - グラフ上でドラッグしてズーム
   - マウスホバーでデータ詳細を表示
   - カメラアイコンで画像を保存

## 次のステップ

### 自分のデータを使う

1. **CSVファイルを準備**
   ```csv
   date,sales,costs
   2024-01-01,100,80
   2024-01-02,105,82
   2024-01-03,98,79
   ```

2. **アップロード**
   - 「📥 CSVファイル選択」ボタンをクリック
   - ファイルを選択

3. **可視化**
   - X軸とY軸を選択
   - グラフタイプを選択

### カスタマイズ

- **カラースキーム**: 好みの配色に変更
- **軸の選択**: 表示するデータ列を変更
- **データ更新**: 「🔄 データ更新」で動的な変化を確認

## よくある質問

### Q1: グラフが表示されない

**A**: ブラウザのコンソールを確認（F12キー）。データに適切な列が含まれているか確認してください。

### Q2: パッケージがインストールできない

**A**: Python 3.8以上が必要です。バージョンを確認:
```bash
python --version
```

### Q3: ポート8050が使用中

**A**: `advanced_graph_app.py`の最終行を変更:
```python
app.run_server(debug=True, host='0.0.0.0', port=8051)  # 別のポート番号
```

## トラブルシューティング

問題が発生した場合:

1. **テストを実行**
   ```bash
   python test_advanced_app.py
   ```

2. **パッケージを再インストール**
   ```bash
   pip install --upgrade -r requirements.txt
   ```

3. **エラーメッセージを確認**
   - ターミナルのエラー出力を確認
   - ブラウザのコンソール（F12）を確認

## 終了方法

アプリを終了するには:

```
Ctrl + C
```

ターミナルで上記のキーを押すとアプリケーションが停止します。
