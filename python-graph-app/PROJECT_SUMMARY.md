# 📊 Python グラフアプリケーション - プロジェクトサマリー

## プロジェクト概要

このプロジェクトには、**2つの異なるグラフアプリケーション**が含まれています：

### 1. 基本版グラフアプリ（tkinter版）
- **ファイル**: `graph_app.py`
- **UI**: デスクトップアプリ（tkinter）
- **特徴**: シンプル、軽量、オフラインで動作

### 2. 高度なグラフアプリ（Web版）
- **ファイル**: `advanced_graph_app.py`
- **UI**: Webブラウザ（Dash）
- **特徴**: 高機能、インタラクティブ、統計分析機能

## 🎯 2つのアプリの比較

| 特徴 | 基本版（tkinter） | 高度版（Dash） |
|------|------------------|---------------|
| **プラットフォーム** | デスクトップアプリ | Webアプリ |
| **起動方法** | `python graph_app.py` | `python advanced_graph_app.py` |
| **グラフタイプ** | 6種類 | 12種類 |
| **インタラクティブ性** | 基本的 | 高度 |
| **統計分析** | なし | あり（回帰、相関） |
| **3Dグラフ** | なし | あり |
| **CSV対応** | あり | あり |
| **リアルタイム更新** | なし | あり |
| **複数ユーザー** | 不可 | 可能 |
| **依存パッケージ** | 少ない | 多い |

## 🚀 どちらを使うべきか？

### 基本版を選ぶべき場合

✅ シンプルなグラフ作成だけが必要
✅ インターネット接続がない環境
✅ 軽量なアプリが好み
✅ デスクトップアプリの方が使いやすい

**起動方法:**
```bash
python graph_app.py
```

### 高度版を選ぶべき場合

✅ 統計分析が必要
✅ 3Dグラフを使いたい
✅ リアルタイムでデータを更新したい
✅ より多くのグラフタイプが必要
✅ Webブラウザで使いたい
✅ 複数人で共有したい

**起動方法:**
```bash
python advanced_graph_app.py
# http://127.0.0.1:8050/ にアクセス
```

## 🔧 トラブルシューティング

### 問題: パッケージがインストールできない

**解決策:**
```bash
# Python バージョン確認（3.8以上が必要）
python --version

# pip を最新に更新
pip install --upgrade pip

# 再度インストール
pip install -r requirements.txt
```

### 問題: 基本版が起動しない（macOS）

**解決策:**
```bash
# tkinter をインストール（通常は不要）
brew install python-tk
```

### 問題: 高度版のポート8050が使用中

**解決策:**
```python
# advanced_graph_app.py の最後の行を変更
app.run_server(debug=True, host='0.0.0.0', port=8051)
```
