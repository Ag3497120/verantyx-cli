# 🎨 高度なグラフアプリケーション - 使用ガイド

## 概要

Plotly と Dash を使用した最先端のインタラクティブデータ可視化Webアプリケーションです。

## 主な特徴

### 🎯 12種類のグラフタイプ

1. **📈 折れ線グラフ** - 時系列データや連続データの可視化
2. **📊 棒グラフ** - カテゴリ別の比較
3. **⭕ 散布図** - 2変数の相関関係
4. **🥧 円グラフ** - 割合の表示
5. **📊 ヒストグラム** - データの分布
6. **📦 箱ひげ図** - 統計的な分布
7. **🔥 ヒートマップ** - 相関行列の可視化
8. **🎯 バブルチャート** - 3変数の関係性
9. **🌐 3D散布図** - 3次元データの可視化
10. **🌊 3D曲面図** - 数学関数の3次元表現
11. **📉 エリアチャート** - 累積データの表示
12. **🎪 バイオリンプロット** - 分布の詳細表示

### 💡 インタラクティブ機能

- **ホバーツールチップ**: マウスをかざすとデータの詳細を表示
- **ズーム・パン**: グラフの拡大縮小と移動
- **データポイント選択**: クリックで特定データを選択
- **凡例の切り替え**: クリックでデータ系列の表示/非表示
- **リアルタイム更新**: データの動的な更新

### 📊 高度な統計分析

1. **基本統計量**
   - 平均値、中央値
   - 標準偏差、分散
   - 最小値、最大値
   - 四分位数

2. **相関分析**
   - 相関係数マトリックス
   - 強い正の相関（>0.7）を緑色でハイライト
   - 強い負の相関（<-0.7）をピンク色でハイライト

3. **回帰分析**
   - 線形回帰
   - 傾き・切片の計算
   - 相関係数（R）
   - 決定係数（R²）
   - p値
   - 標準誤差
   - 回帰直線の可視化

### 💾 データ管理機能

- **CSVインポート**: ドラッグ&ドロップでCSVファイルを読み込み
- **CSVエクスポート**: 現在のデータをCSV形式で保存
- **サンプルデータ**: ワンクリックでサンプルデータを生成
- **リアルタイム更新**: データを動的に変更して変化を観察

## インストール方法

### 1. 必要なパッケージをインストール

```bash
cd python-graph-app
pip install -r requirements.txt
```

または個別にインストール:

```bash
pip install dash dash-bootstrap-components plotly pandas scipy numpy
```

### 2. アプリケーションを起動

```bash
python advanced_graph_app.py
```

または実行権限を付与して:

```bash
chmod +x advanced_graph_app.py
./advanced_graph_app.py
```

### 3. ブラウザでアクセス

```
http://127.0.0.1:8050/
```

## 使い方

### 基本操作

1. **グラフタイプの選択**
   - 左パネルの「グラフの種類」ドロップダウンから選択
   - 自動的にグラフが更新されます

2. **軸の選択**
   - X軸とY軸に表示する列を選択
   - データ列から自動的に選択肢が生成されます

3. **カラースキームの変更**
   - 「カラースキーム」から好みの配色を選択
   - ビビッド、プラズマ、クール、ウォームなど

### データ操作

#### サンプルデータを読み込む

「📁 サンプルデータ読込」ボタンをクリック
- 100日分の売上、コスト、利益データが生成されます
- カテゴリ別、地域別のデータも含まれます

#### CSVファイルをインポート

1. 「📥 CSVファイル選択」ボタンをクリック
2. CSVファイルを選択
3. 自動的にデータが読み込まれます

**CSVファイルの形式例:**

```csv
date,sales,costs,profit,category,region
2024-01-01,100,80,20,A,北
2024-01-02,105,82,23,B,南
2024-01-03,98,79,19,C,東
```

#### データをエクスポート

「📤 CSV エクスポート」ボタンをクリック
- 現在のデータがCSV形式でダウンロードされます
- ファイル名は自動的に日時が付与されます: `graph_data_YYYYMMDD_HHMMSS.csv`

#### データを更新

「🔄 データ更新」ボタンをクリック
- データに小さな変動が追加されます
- リアルタイム更新のシミュレーション

### 統計分析

#### 基本統計を表示

「📊 基本統計」ボタンをクリック
- 数値列の統計量が表形式で表示されます
- 平均、標準偏差、最小値、最大値など

#### 相関分析を実行

「🔗 相関分析」ボタンをクリック
- すべての数値列間の相関係数マトリックスを表示
- 強い相関は色付きでハイライト

#### 回帰分析を実行

「📈 回帰分析」ボタンをクリック
- 選択されたX軸とY軸のデータで線形回帰を実行
- 回帰直線のグラフと統計情報を表示

### グラフの操作

#### ズーム

- **ドラッグでズーム**: グラフ上をドラッグして領域を選択
- **ダブルクリック**: ズームをリセット

#### パン（移動）

- ツールバーの「パン」アイコンをクリック後、ドラッグして移動

#### グラフの保存

- カメラアイコンをクリック
- PNG形式で高解像度画像（1920x1080、2倍スケール）を保存

#### ホームに戻る

- 「ホーム」アイコンをクリックでデフォルトビューに戻る

## 各グラフタイプの詳細

### 📈 折れ線グラフ

**用途**: 時系列データ、トレンドの可視化

**操作**:
- X軸: 通常は日付や時間
- Y軸: 数値データ
- マーカーと線の組み合わせで表示

### 📊 棒グラフ

**用途**: カテゴリ別の比較

**特徴**:
- 値に応じて色が変化（カラーマップ使用）
- ホバーで正確な値を表示

### ⭕ 散布図

**用途**: 2変数の相関関係の分析

**特徴**:
- カテゴリで色分け
- サイズで数量を表現
- すべての列のデータをホバーで表示

### 🥧 円グラフ

**用途**: 全体に対する割合の表示

**特徴**:
- カテゴリごとに自動集計
- パーセンテージ表示
- インタラクティブなセグメント選択

### 📊 ヒストグラム

**用途**: データの分布の可視化

**特徴**:
- 自動的にビン（区間）を計算
- カテゴリ別に色分け可能
- 正規分布の確認などに有用

### 📦 箱ひげ図

**用途**: データの統計的分布の表示

**表示される情報**:
- 中央値（箱の中の線）
- 第1四分位数と第3四分位数（箱の両端）
- 外れ値（点）
- 最小値・最大値（ひげ）

### 🔥 ヒートマップ

**用途**: 相関行列の可視化

**特徴**:
- すべての数値列間の相関を一目で把握
- 赤=正の相関、青=負の相関
- カラースケールのカスタマイズ可能

### 🎯 バブルチャート

**用途**: 3つの変数の関係性を表示

**軸**:
- X軸: 売上
- Y軸: コスト
- バブルサイズ: 数量
- 色: カテゴリ

### 🌐 3D散布図

**用途**: 3次元データの可視化

**特徴**:
- X, Y, Z軸で3つの数値を表示
- マウスで回転・ズーム可能
- カテゴリで色分け

### 🌊 3D曲面図

**用途**: 数学関数や地形データの可視化

**特徴**:
- 連続的な曲面を表示
- カラーマップで高さを表現
- 回転して任意の角度から観察

### 📉 エリアチャート

**用途**: 累積データや時系列の変化

**特徴**:
- 折れ線の下が塗りつぶされる
- 複数系列の積み重ね表示

### 🎪 バイオリンプロット

**用途**: データ分布の詳細な可視化

**特徴**:
- 箱ひげ図とカーネル密度推定の組み合わせ
- データの分布形状が一目でわかる
- すべてのデータポイントも表示可能

## カラースキームガイド

### デフォルト (plotly)
- Plotlyの標準カラーパレット
- バランスの取れた配色

### ビビッド (viridis)
- 紫から黄色のグラデーション
- 色覚異常にも配慮した配色
- 科学的可視化に適する

### プラズマ (plasma)
- 濃い紫から明るい黄色
- コントラストが強い
- ダイナミックなデータ表現

### インファーノ (inferno)
- 黒から黄色の火炎調
- 熱や強度の表現に適する

### クール (blues)
- 青系のグラデーション
- 落ち着いた印象
- 水や冷たさの表現

### ウォーム (reds)
- 赤系のグラデーション
- 温かい印象
- 熱や重要度の表現

## トラブルシューティング

### アプリが起動しない

```bash
# パッケージを再インストール
pip install --upgrade dash dash-bootstrap-components plotly pandas scipy numpy

# Pythonのバージョン確認（3.8以上が必要）
python --version
```

### ポート8050が使用中

別のポート番号で起動:

```python
# advanced_graph_app.py の最後の行を変更
app.run_server(debug=True, host='0.0.0.0', port=8051)  # 8050 → 8051
```

### CSVファイルが読み込めない

- ファイルがUTF-8エンコーディングか確認
- 1行目にヘッダー（列名）があるか確認
- カンマ区切りになっているか確認

**エンコーディング変換例:**

```bash
# Shift-JIS から UTF-8 に変換（macOS/Linux）
iconv -f SHIFT-JIS -t UTF-8 input.csv > output.csv
```

### グラフが表示されない

- ブラウザのコンソールでエラーを確認（F12キー）
- データに適切な列が含まれているか確認
- 数値データが必要なグラフで文字列データを選んでいないか確認

### メモリエラーが発生

大きなCSVファイルの場合、データをサンプリング:

```python
# CSVの最初の10000行だけ読み込む
df = pd.read_csv('large_file.csv', nrows=10000)
```

## 高度な使用例

### 例1: 株価データの分析

```python
# データ準備
import yfinance as yf

# 株価データ取得
ticker = yf.Ticker("AAPL")
df = ticker.history(period="1y")
df = df.reset_index()

# CSVとして保存
df.to_csv('stock_data.csv', index=False)

# アプリで読み込み
# 折れ線グラフでCloseを表示
# ボリュームは棒グラフで表示
```

### 例2: 気象データの可視化

```csv
date,temperature,humidity,pressure,weather
2024-01-01,5.2,65,1013,晴れ
2024-01-02,6.8,70,1015,曇り
2024-01-03,4.5,80,1010,雨
```

- 折れ線グラフで気温の推移
- ヒートマップで変数間の相関
- 散布図で気温と湿度の関係

### 例3: 売上分析

```csv
date,product,sales,costs,region,category
2024-01-01,製品A,1000,700,東京,電化製品
2024-01-01,製品B,800,500,大阪,家具
```

- 棒グラフで製品別売上
- 円グラフで地域別シェア
- バブルチャートで売上・コスト・数量の関係
- 回帰分析で売上予測

## パフォーマンス最適化

### 大量データの扱い

1. **データのサンプリング**
   - 10万行以上のデータは適度にサンプリング
   - 代表的なデータポイントを抽出

2. **集計してから可視化**
   - 日次データを週次・月次に集計
   - グループ化して平均値を表示

3. **必要な列だけ読み込む**
   ```python
   df = pd.read_csv('data.csv', usecols=['date', 'sales', 'profit'])
   ```

## カスタマイズ

### グラフの高さを変更

`advanced_graph_app.py` の `dcc.Graph` コンポーネント:

```python
style={'height': '700px'}  # 500px → 700px
```

### カラーテーマを変更

```python
external_stylesheets=[dbc.themes.DARKLY]  # ダークテーマ
# または
external_stylesheets=[dbc.themes.COSMO]   # ライトテーマ
```

利用可能なテーマ:
- BOOTSTRAP, CERULEAN, COSMO, CYBORG
- DARKLY, FLATLY, JOURNAL, LITERA
- LUMEN, LUX, MATERIA, MINTY
- MORPH, PULSE, QUARTZ, SANDSTONE
- SIMPLEX, SKETCHY, SLATE, SOLAR
- SPACELAB, SUPERHERO, UNITED, VAPOR
- YETI, ZEPHYR

### サンプルデータをカスタマイズ

`generate_sample_data()` 関数を編集:

```python
def generate_sample_data():
    # あなた独自のデータ生成ロジック
    df = pd.DataFrame({
        'month': ['1月', '2月', '3月', '4月', '5月', '6月'],
        'revenue': [100, 150, 130, 180, 220, 250],
        'expenses': [80, 90, 85, 100, 120, 130]
    })
    return df
```

## API リファレンス

### 主要な関数

#### `generate_sample_data()`
サンプルデータを生成

**戻り値**: pandas DataFrame

#### `update_graph(graph_type, x_col, y_col, color_scheme, animation, data)`
グラフを更新

**引数**:
- `graph_type`: グラフの種類
- `x_col`: X軸の列名
- `y_col`: Y軸の列名
- `color_scheme`: カラースキーム
- `animation`: アニメーション設定
- `data`: データ（辞書形式）

**戻り値**: Plotly figure オブジェクト
