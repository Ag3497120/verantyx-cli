# インタラクティブテストガイド
Interactive Developmental AI Test Guide

---

## 🎮 概要

**あなたが直接、発達段階AIを体験できるインタラクティブツール**です。

- 画像や動画を処理して、AIの反応を確認
- 年齢を手動で設定して、各発達段階を体験
- リアルタイムで能力の変化を確認

---

## 🚀 起動方法

### **基本起動**

```bash
cd /Users/motonishikoudai/verantyx_v6/verantyx-cli
python3 verantyx_cli/vision/interactive_test.py
```

## 📊 おすすめテストシナリオ

### **シナリオ1: 発達段階の確認**

```bash
1. 年齢を0歳に設定
2. 画像を処理 → 結果を確認
3. 年齢を1歳に設定
4. 同じ画像を処理 → 空間情報の追加を確認
5. 年齢を3歳に設定
6. 2枚の画像を連続処理 → 移動軌跡を確認
7. 年齢を7歳に設定
8. 複数画像を処理 → カテゴリ分類を確認
```

### **シナリオ2: 動画学習**

```bash
1. 年齢を1歳に設定（空間記憶を有効化）
2. 動画を処理（10-20フレーム）
3. 状態を確認 → 経験数の増加を確認
4. 別の動画を処理
5. 予測誤差の変化を観察
```

### **シナリオ3: カテゴリ学習**

```bash
1. 年齢を7歳に設定
2. 似た画像を3枚処理（例: 猫の画像3枚）
   → 同じカテゴリIDが付与される
3. 全く違う画像を処理（例: 車の画像）
   → 異なるカテゴリIDが付与される
```

## 💡 ヒント

### **1. 画像サイズは問わない**

- どんなサイズの画像でもOK
- 自動的に64x64にリサイズされる

### **2. 動画は短めに**

- 処理フレーム数: 10-20フレーム推奨
- 多すぎると時間がかかる

### **3. 同じ画像で比較**

- 0歳と1歳で同じ画像を処理すると、違いが明確

### **4. 連続処理で軌跡確認**

- 3歳で画像を2枚連続処理すると、移動軌跡が見える

### **5. カテゴリは複数枚必要**

- 7歳で類似画像を複数処理すると、カテゴリ分類が動作

---

## ⚠️ 注意事項

### **1. デーモンとの干渉**

このツールは独立したインスタンス。バックグラウンドのデーモンには影響しません。

### **2. データは保存されない**

インタラクティブテストの記憶は、終了すると消えます。

### **3. 処理速度**

- 画像1枚: 約1-2秒
- 動画10フレーム: 約10-20秒
