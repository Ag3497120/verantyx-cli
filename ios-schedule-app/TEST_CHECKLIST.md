# 動作確認チェックリスト

アプリが正しく動作しているか確認するためのチェックリストです。

## 基本機能

### 初期表示
- [ ] アプリが起動する
- [ ] カレンダーが表示される
- [ ] 今日の日付がハイライトされている
- [ ] サンプルスケジュールが3件表示される
- [ ] 検索バーが表示される
- [ ] 右上に「+」ボタンがある

### カレンダー操作
- [ ] 左矢印「<」で前月に移動できる
- [ ] 右矢印「>」で次月に移動できる
- [ ] 日付をタップすると選択状態になる（青背景）
- [ ] 日曜日が赤色、土曜日が青色で表示される
- [ ] スケジュールがある日に青い点が表示される
- [ ] 日付を選択すると、その日のスケジュールが表示される

## スケジュール追加

### 新規作成画面
- [ ] 「+」ボタンをタップすると追加画面が開く
- [ ] タイトル入力欄がある
- [ ] 説明入力欄がある
- [ ] カテゴリ入力欄がある
- [ ] 場所入力欄がある
- [ ] 日付選択ができる
- [ ] 開始時刻選択ができる
- [ ] 終了時刻選択ができる
- [ ] 優先度を3段階（高・中・低）から選択できる
- [ ] 「キャンセル」ボタンで閉じることができる

### バリデーション
- [ ] タイトルを空欄で保存しようとするとエラーメッセージが表示される
- [ ] 終了時刻を開始時刻より前に設定するとエラーメッセージが表示される

### 保存機能
- [ ] 「保存」ボタンをタップすると、メイン画面に戻る
- [ ] 追加したスケジュールがリストに表示される
- [ ] カレンダー上の該当日に青い点が追加される

## スケジュール表示

### リスト表示
- [ ] スケジュールが時刻順に並んでいる
- [ ] タイトルが表示される
- [ ] 時間範囲（開始〜終了）が表示される
- [ ] カテゴリが青いラベルで表示される
- [ ] 優先度が色付きラベルで表示される（高=赤、中=オレンジ、低=緑）
- [ ] 左側に完了チェック用の○ボタンがある

### 詳細画面
- [ ] スケジュールをタップすると詳細画面が開く
- [ ] タイトルが表示される
- [ ] 優先度が表示される
- [ ] 日付が表示される
- [ ] 時間が表示される
- [ ] カテゴリが表示される
- [ ] 場所が表示される
- [ ] 説明が表示される
- [ ] 右上に「編集」ボタンがある
- [ ] 右上に完了チェックボタンがある
- [ ] 下部に「削除」ボタンがある

## スケジュール編集

### 編集モード
- [ ] 「編集」ボタンをタップすると編集モードになる
- [ ] 全ての項目が編集可能になる
- [ ] 「保存」ボタンで変更が保存される
- [ ] 「キャンセル」ボタンで変更が破棄される

### 更新反映
- [ ] 編集内容がリストに反映される
- [ ] 日付を変更すると、カレンダー上の表示が更新される

## スケジュール削除

### 削除方法1: スワイプ
- [ ] リスト上でスケジュールを左にスワイプできる
- [ ] 「Delete」ボタンが表示される
- [ ] タップすると削除される

### 削除方法2: 詳細画面
- [ ] 詳細画面の「削除」ボタンをタップすると確認ダイアログが表示される
- [ ] 「キャンセル」で削除をキャンセルできる
- [ ] 「削除」でスケジュールが削除される
- [ ] 削除後、メイン画面に戻る

## 完了チェック機能

### チェック操作
- [ ] リストの左側の○ボタンをタップすると✓マークになる
- [ ] スケジュールが半透明になる
- [ ] タイトルに取り消し線が引かれる
- [ ] もう一度タップすると未完了に戻る

### 詳細画面でのチェック
- [ ] 詳細画面右上のボタンでもチェックできる
- [ ] チェック状態がリストに反映される

## 検索機能

### 検索動作
- [ ] 検索バーに文字を入力できる
- [ ] タイトルで検索できる
- [ ] 説明で検索できる
- [ ] カテゴリで検索できる
- [ ] 検索結果がリアルタイムで絞り込まれる
- [ ] 検索文字列を削除すると全スケジュールが表示される
- [ ] 検索結果が0件の場合、適切なメッセージが表示される

## データ永続化

### 保存・復元
- [ ] スケジュールを追加して、アプリを完全に終了
- [ ] アプリを再起動すると、追加したスケジュールが残っている
- [ ] スケジュールを編集して、アプリを終了・再起動
- [ ] 編集内容が保持されている
- [ ] スケジュールを削除して、アプリを終了・再起動
- [ ] 削除が保持されている
- [ ] 完了チェックの状態が保持されている

## UI/UX

### デザイン
- [ ] 日本語で表示されている
- [ ] フォントサイズが読みやすい
- [ ] 色分けが見やすい（優先度、カテゴリ）
- [ ] タップ領域が適切な大きさ
- [ ] スクロールがスムーズ

### ナビゲーション
- [ ] 画面遷移がスムーズ
- [ ] 戻るボタン（<）が機能する
- [ ] モーダルを閉じることができる

## エッジケース

### 空の状態
- [ ] スケジュールがない日を選択すると「スケジュールがありません」と表示される
- [ ] 全スケジュールを削除すると空の状態メッセージが表示される

### 大量データ
- [ ] スケジュールを10件以上追加してもスムーズに動作する
- [ ] スクロールが正常に動作する

### 日付の境界
- [ ] 月の境界（月末から月初）でカレンダーが正しく表示される
- [ ] 年の境界（12月から1月）で正しく動作する

---

## チェック完了後

すべてのチェック項目にチェックが入れば、アプリは正常に動作しています！

問題があった項目は、エラーメッセージや動作を記録して、トラブルシューティングを参照してください。
