# iOS スケジュールアプリ

SwiftUIで作成したシンプルで使いやすいスケジュール管理アプリです。

## 主な機能

### 1. カレンダー表示
- 月間カレンダービュー
- 日付選択機能
- スケジュールがある日にマーカー表示
- 今日の日付をハイライト表示

### 2. スケジュール管理
- スケジュールの追加、編集、削除
- タイトル、説明、カテゴリ、場所の設定
- 開始時刻と終了時刻の設定
- 3段階の優先度設定（高・中・低）
- 完了チェック機能

### 3. 検索機能
- タイトル、説明、カテゴリでの検索

### 4. データ永続化
- UserDefaultsを使用したデータの自動保存
- アプリを閉じても データが保持されます

## ファイル構成

```
ScheduleApp/
├── ScheduleAppApp.swift          # アプリのエントリポイント
├── Schedule.swift                # データモデル
├── ScheduleViewModel.swift       # ビジネスロジック
├── ContentView.swift             # メインビュー（カレンダー + リスト）
├── AddScheduleView.swift         # スケジュール追加画面
└── ScheduleDetailView.swift      # スケジュール詳細・編集画面
```

## 使い方

### Xcodeでプロジェクトを開く

1. Xcodeを起動
2. "Create a new Xcode project" を選択
3. "iOS" > "App" を選択
4. プロジェクト名: `ScheduleApp`
5. Interface: `SwiftUI`
6. Language: `Swift`
7. このディレクトリ内のSwiftファイルをプロジェクトに追加

### ビルドと実行

1. シミュレータまたは実機を選択
2. ⌘ + R でビルド・実行

## 主な画面

### メイン画面
- 上部: 月間カレンダー
- 中央: 検索バー
- 下部: 選択した日のスケジュール一覧
- 右上の + ボタンで新規スケジュールを追加

### 新規スケジュール画面
- タイトル（必須）
- 説明
- カテゴリ
- 場所
- 日付と時刻
- 優先度（3段階）

### 詳細画面
- スケジュールの詳細表示
- 編集ボタンで編集モードに切り替え
- 完了チェックボタン
- 削除ボタン

## 技術仕様

- **言語**: Swift
- **フレームワーク**: SwiftUI
- **最低バージョン**: iOS 15.0以上
- **データ永続化**: UserDefaults
- **状態管理**: @StateObject, @ObservedObject
- **ナビゲーション**: NavigationView

## カスタマイズ

### 優先度の色を変更
`Schedule.swift` の `Priority` enum内の `color` プロパティを編集

### カレンダーの曜日表記を変更
`ContentView.swift` の `CalendarView` 内の `daysOfWeek` 配列を編集

### サンプルデータの変更
`Schedule.swift` の `sampleData` プロパティを編集
