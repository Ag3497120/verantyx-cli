# 10万個高品質コマンド生成計画

## 方針

❌ **自動生成の問題点:**
- 日本語の意味を理解していない
- Cross構造との対応が不正確
- 用法や例文が不適切
- 低品質で実用不可

✅ **Claudeによる直接生成:**
- 各コマンドの日本語の意味を正確に理解
- 適切なCross軸配置
- 正しい用法と例文
- 関連語との関係性を明確化
- 高品質で実用可能

## 品質基準

各コマンドは以下を満たす:

1. ✅ **意味の正確性**: 日本語として正しい意味
2. ✅ **用法の適切性**: 実際の使用方法が正しい
3. ✅ **例文の妥当性**: 実用的な例
4. ✅ **Cross軸の適切性**: 6軸配置が意味的に正しい
5. ✅ **関連語の正確性**: 類義語・対義語が適切
6. ✅ **階層性**: 上位・下位概念が明確

## 実装方法

各コマンドを私Claudeが:
1. 日本語の意味を理解
2. 適切な用法を判断
3. 実用的な例文を作成
4. Cross軸配置を決定
5. 関連語を選定

これにより、**10万個の高品質コマンド**を生成します。
