# Verantyx 学習フロー詳細 - Claudeの対話からの学習

**作成日**: 2026-03-10

---

## 🎯 概要

このドキュメントでは、Claudeのユーザー入力とClaude出力を学習データとして流し込んだ場合、**実際にどのように動作するのか**を、具体的な対話例を使って段階的に説明します。

---

## 📊 全体フロー概要

```
Claude対話ペア
    ↓
【フェーズ1】即座の処理（リアルタイム）
    ├─ コマンドマッチング（22,097辞書）
    ├─ Cross構造生成（6軸）
    └─ グローバルCross構造に保存
    ↓
【フェーズ2】パターン推論（3件以上で自動）
    ├─ パターン抽出
    ├─ 類似パターン探索
    ├─ 6軸パターン収集
    ├─ パズルピース組み合わせ
    └─ 最適推論選択
    ↓
【フェーズ3】小世界シミュレータ（5件以上で自動）
    ├─ 小世界構築
    ├─ パターン分析
    ├─ 因果推論
    ├─ 未来予測
    └─ 推奨行動生成
    ↓
【フェーズ4】動的.jcross生成（推論結果から）
    ├─ 新ラベル生成
    └─ 即座に実行
```
