## Cross構造の活用

### 6軸フル活用

すべての.jcross実装でCross構造の6軸を完全に活用:

1. **UP軸**: 抽象レベル（メタ→戦略→実行）
2. **DOWN軸**: 具体レベル（生データ、詳細）
3. **FRONT軸**: 未来（予測、目標）
4. **BACK軸**: 過去（記憶、履歴、蓄積）
5. **RIGHT軸**: 展開（次の状態、関連）
6. **LEFT軸**: 基盤（前提、依存）

### Cross操作の例

```jcross
# Cross構造に記録
実行する cross.配列追加 学習データCross "BACK" 0 "記憶" 新規データ

# Cross構造から検索
実行する cross.検索 日本語語彙Cross "単語"

# Cross値を増加
実行する cross.増加 対話パターンCross "DOWN" 1 "頻度" 1
```
