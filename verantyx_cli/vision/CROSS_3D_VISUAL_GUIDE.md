# Cross構造の3D視覚表現ガイド
Cross Structure 3D Visual Representation Guide

## 目的

20,000点以上のCross構造を「立体的にどう見えるか」を視覚的に理解するためのガイド。
横・縦・前後の各方向から見たときの点の配置パターンを図解する。

---

```
    ●                  ●                 ●
   ●●●      →        ●●●      →       ●●●
    ●                  ●                 ●
```

```
【X軸回転（縦回転）】
```
